# Global Issue Explorer Documentation — v1.0.0

## 🚀 Initial Release

Welcome to the first official release of **Global Issue Explorer**! This tool provides a powerful, client-side interface to browse, filter, and sort GitHub issues across the entire platform.

## ✨ Core Functionality

* **Cross-Repo Search:** Break free from repository limits and explore issues across all of GitHub.
* **Customizable Views:** Filter by language, labels, and status to find exactly what you need.
* **Client-Side Efficiency:** Fully client-side processing for speed and privacy.
* **Rate-Limit Optimized:** Built-in support for GitHub Personal Access Tokens (PATs) to maximize your API request limits.

## 🛠️ Getting Started

1. **Prerequisites:** Ensure you have [[Node.js](https://nodejs.org/)](https://nodejs.org/) installed.
2. **Setup:** Run `npm install` to grab dependencies.
3. **Run:** Execute `npm run dev` to start the interface.
4. **Configuration:** Add your `GITHUB_TOKEN` to a `.env.local` file to move from 10 to 30 requests per minute. See the `README.md` for details on how to generate your fine-grained token.
